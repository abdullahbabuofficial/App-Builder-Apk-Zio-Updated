# Cursor MCP (optional)

- Edit **`mcp.json`** and replace **`YOUR_PROJECT_REF`** with your Supabase project ref (Dashboard → Project Settings → General).
- Reload Cursor; complete the Supabase login/auth prompt when the MCP connects.
- Run **`make supabase-env-hints SUPABASE_PROJECT_REF=<ref>`** from the repo root for matching `VITE_*` / `DATABASE_URL` patterns.
